  <h2><code>$ whoami</code></h2>
</div>

> **Building at the intersection of Web3 and AI** -- where smart contracts meet autonomous agents,
> and protocol engineering meets agentic systems.
>
> CTO at **[Integra Foundation](https://integralayer.com)**, shipping a real-estate RWA Layer-1
> from genesis block to production. Simultaneously building AI agent ecosystems that automate
> market intelligence, code review, content generation, and engineering operations.

I co-authored **3 Ethereum standards** ([ERC-3643](https://eips.ethereum.org/EIPS/eip-3643),
[ERC-6960](https://eips.ethereum.org/EIPS/eip-6960),
[ERC-7410](https://eips.ethereum.org/EIPS/eip-7410)) adopted across DeFi and security token
ecosystems. Led the Nexus smart accounts team at [Biconomy](https://github.com/bcnmy)
(**38M txns, $441M volume, $5.6M saved**), built CeDeFi platforms locking **$20M+ TVL**,
and deployed **2,000+ smart contracts** across Ethereum, Polygon, Fantom, BSC, and Avalanche.

On the AI side, I build **multi-agent systems** with self-learning loops, **MCP server** integrations,
and **Claude Code plugin ecosystems** -- from prediction market agents and autonomous trading bots
to agentic engineering workflows that ship production code end-to-end.
